*******
install sigc
*******

Regular install
===============

To install sigc run:

.. code-block:: bash

    pip install sigc

run this pip command to specify the version and installation source:

.. code-block:: bash

    pip install -i https://pypi.org/simple sigc==0.1.17

Checking version
================

To check your sigc version

.. code-block:: python

    import sigc
    sigc.__version__
